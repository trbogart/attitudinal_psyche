To run the script in interactive mode, install Python and run
`python ap_shadow_type_calculator`

To run the script with arguments, run
`python ap_shadow_type_calculator [ap_type] [subtype]`

AP type is any of the 24 permutation of FLEV (e.g. LFEV and LEFV).
Subtype is 4 digits between 0 and 4.
